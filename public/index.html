KushAI Plugins Marketplace

Central repository for KushAI plugins

API Endpoints

Get All Plugins

GET https://moto.github.io/KushAI-Plugins/plugins.json

Returns a list of all available plugins in the marketplace.

Get Plugin Details

GET https://moto.github.io/KushAI-Plugins/plugins/{plugin-id}.json

Returns detailed information about a specific plugin.

Plugin Structure

Each plugin in the marketplace follows a specific structure:

  • Plugin metadata in plugin.json
  • Plugin code and assets in the plugin's directory

How to Use

To use plugins from this marketplace:

  1. Browse available plugins in the plugins directory
  2. Download and install plugins manually, or
  3. Use the KushAI application to automatically discover and install plugins
